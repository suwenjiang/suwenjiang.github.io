Touchsceen not working in SurfacePro Fedora

$ published 2022-08-12 · updated 2026-07-24 · #Surface Pro #Fedora

Backgroud

Having a low spec surfacepro 6 with windows 10 installed which is get delayed response.
So got the idea to change windows 10 to a Linux system. After googled a lot, Fedora seems the most recommended distro for surface pro.

However, when i installed the fedora36 by folling the link
.. Initiall things look perfect, howver, not too sure what i did later,
the touchscreen not responsing at all.

Soulution

Here are the steps:

  1. Remove the surcebook package

    sudo dnf remove linux-surface-secureboot-mok
    not just uninstall but remove, then reboot the computer and follow the screen to enter password ‘surface’ is the default to delete associated certificate.

  2. reinstall secureboot

    sudo dnf install linux-surface-secureboot-mok

  3. run command

    sudo grubby –set-default /boot/vmlinuzsurface

  4. reboot computer

About steps should get touchsceen functioning
