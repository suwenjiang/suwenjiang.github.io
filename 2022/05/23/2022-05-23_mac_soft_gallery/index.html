mac上安装的软件

常用软件类

  • Alfread

一开始只把它简单的作为一个程序启动器，但是这就和mac自带的spotlight没啥区别了。但是购买了powerpack使用上了workflow后，真是神器啊。

当年在Windows平台上，hotkey和wox就让我惊喜不已。Alfread真不知比他们强多少啊。

  • qq
  • mpv

在window端一直使用potplay和qq影音的。potplay没有mac断的，使用mpv强大的播放器。

  • LICEcap

一直使用的用来录制gif图片的工具，windows和mac端都有。

  • 图片查看类

开发类

  • Dash

对于一个开发人员来说，经常需要查询开发接口等。Dash里面集合各种开发语言文档，常见软件的cheatsheet，atom，git等等。

牛逼的地方是Dash支持和各种IDE和软件结合使用。可以直接在这IDE和软件中查询Dash文档

  • visual studio code
    由奢入简难啊。编程以VS起家，后面前前后后使用过许多的IDE，都么有VS好用。vs code 私以为还是个
    文本编辑器吧，类sublime的文本编辑器。sublime作为个文本编辑器我一直都不感冒，感觉作为个文本编辑
    器就应该向nodepad++ 那样的简单。

  • webstorm

  • pycharm

  • nodejs

  • mongodb

  • mongochef

尝试过的mongodb的client端中，一款非常简单易用的mongodb数据库客户端软件，在各个平台上都有对应的安装文件

go2shell

在当前的目录下启动shell,支持terminal, iterm

进入preference

open -a Go2Shell –args config

清理类

Dr. Cleaner 一个免费好用的清理软件。当软件卸载后，能够自动的清除残余文件。
