windows docker mount shared directory

$ published 2017-10-29 · updated 2026-07-21 · #docker #shared directory

In docker for windows:

  • Mount d:/logs directory into ubuntu contain as /data

docker run -it -v d:/Logs:/data ubuntu /bin/sh

or

docker run -it -v d:/Logs:/data ubuntu bash

  • Mount directory to nginx

    docker run –name blogserver -v D:/Dropbox/Dropbox/blog/personal_blog/suwenjiang.github.com:/usr/share/nginx/html:ro -d -p 7777:80 nginx
